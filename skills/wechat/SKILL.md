## 从"AGI智能纪元"学到的写作风格

### 1. 标题写作策略（Hook）
- **直击痛点**：如"拒绝流水账"、"别再光收藏不产出"
- **利益点明确**：强调"自动化"、"复利模型"、"终极指南"
- **私域感**：多用"我的..."，增加真实性和经验背书
- **数字驱动**：如"5种方案"、"30分钟"
- **系列感**：用"01/02/03"或"第一期/第二期"

### 2. 内容定位
- **高度垂直**：连续围绕一个主题（Obsidian）
- **系列化**：形成完整知识体系（入门→进阶→高级）
- **从底层到实操**：既有原理讲解，也有实操指南
- **解决具体问题**：撰写系统性指南

## 从用户Obsidian库学到的写作风格

### 文章结构分析

**1. 开场白**
- 第一人称叙述："大家好，我是个重度笔记上瘾患者"
- 痛点引入：用亲身经历建立共鸣
- 明确目标读者

**2. 亲身故事线**
- 时间线叙事：2018印象笔记 → 2021 Notion → 2024 Obsidian
- 具体场景：开会卡15秒的尴尬经历
- 情感表达："那一刻，我知道自己回不去了"

**3. 对比技巧**
- 表格/列表对比Notion vs Obsidian
- 具体数据：5000+页面加载慢
- 诚实缺点：学习曲线陡、协作弱

**4. 实战干货**
- Vault结构展示
- 迁移步骤清单
- 插件推荐
- 代码模板

**5. 互动结尾**
- 总结优点
- 诚实说缺点（增加可信度）
- 互动提问："你呢？还在Notion还是印象笔记挣扎？"
- 固定结尾：下期预告 + 点在看转发

### 写作模板

```markdown
## 引言/痛点引入
上一期...发出后，大家...
但随之而来的是...

## 第一步/核心内容
### XXX（功能/技巧）
**为什么必装/为什么重要**
**使用方法/步骤**
[截图]

## 进阶玩法
...

## 写在最后
核心观点总结
下期预告
固定互动：点个在看+转发
```

### 写作特点总结

| 特点 | 说明 |
|------|------|
| 第一人称 | 真实个人经历 |
| 故事线 | 从过去到现在的发展 |
| 系列化 | 形成完整知识体系 |
| 细节丰富 | 具体年份、数据、场景 |
| 干货充足 | 结构、插件、迁移步骤、代码模板 |
| 诚实可信 | 主动说缺点 |
| 互动结尾 | 固定格式：下期预告+求转发 |
| 对比表格 | 方案对比、价格对比 |

## 文章类型

1. **教程干货**（如"终极指南"）- 最受欢迎，易获赞赏
2. **痛点+解决方案**（如"别再XXX"）
3. **经验复盘**（如"我的XXX"）
4. **对比评测**（如"XXX vs XXX"）
5. **系列合集**（如"01/02/03"）

## 写作流程

1. 确定痛点/主题
2. 收集素材（官方文档、实践经验）
3. 列出大纲（从原理到实操）
4. 撰写初稿
5. 优化标题（痛点+利益点）
6. 配图排版
7. 添加下期预告
